About Medical Perception

Medical Perception is a demo web application that helps users understand common symptoms and possible remedies. It allows you to enter symptoms and receive general health suggestions instantly.

⚕️ This website was built using HTML, CSS, and JavaScript — no backend or database is used. All user information (like login credentials) is stored securely within your own browser using localStorage.

This project was created for educational purposes to demonstrate basic web technologies, user authentication, and form handling. It should not be used for real medical diagnosis or treatment.

Always consult a certified healthcare professional for accurate diagnosis and treatment.
